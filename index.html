Short overview

I work on securing remote monitoring and control systems built on IoT and IIoT technologies, focusing on experimental testbeds, ML-based intrusion detection, and Zero Trust architectures.

  • Design and operation of an IIoT MQTT testbed (ESP32, Raspberry Pi, Node-RED, InfluxDB, Grafana).
  • Real-time anomaly detection for MQTT traffic using ML models deployed on edge devices.
  • Application of identity-aware and policy-enforced architectures to remote labs and ICS environments.
